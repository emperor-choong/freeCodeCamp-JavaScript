# Odd or Even?  
## [13 January 2026]

Given a positive integer, return `"Odd"` if it's an odd number, and `"Even"` is it's even.

**Tests**
1. `oddOrEven(1)` should return `"Odd"`.
2. `oddOrEven(2)` should return `"Even"`.
3. `oddOrEven(13)` should return `"Odd"`.
4. `oddOrEven(196)` should return `"Even"`.
5. `oddOrEven(123456789)` should return `"Odd"`.
